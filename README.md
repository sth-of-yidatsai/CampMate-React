## ⛺ 一站式露營電商平台 - 前台開發
 
專案使用 **React** 進行開發，CampMate 為露營愛好者打造的一站式平台，主要提供露營用品租借、露營區訂房與團露等服務。

在專案中，我負責 **UI視覺設計** 、 **首頁** 、 **領取優惠券專區** 與 **會員中心-我的優惠券** 的切板以及功能開發。


※ 此專案僅供資策會前端工程師就業養成班大專展示使用。
